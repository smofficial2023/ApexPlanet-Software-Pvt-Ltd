## 💡 Key Learnings

1. **Real-world data is always messy** — I learned to audit before cleaning, and document every transformation made.
2. **SQL is the analyst's superpower** — Window functions and CTEs unlocked insights without needing Python at all.
3. **Dashboards are conversations** — Great dashboards anticipate what a manager will ask next and let them explore independently.
4. **Insights need context** — Every number needs a "so what?" framed in terms of revenue, risk, or opportunity.
5. **Documentation is a deliverable** — Code without comments and reports without methodology are incomplete work.
6. **Iteration beats perfection** — Showing rough work early and refining with feedback consistently produced better outcomes.
